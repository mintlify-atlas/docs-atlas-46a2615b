## Overview

Daikin Etiquetas features a custom-styled Windows Forms interface with a borderless window, dark theme, and interactive components. All UI components are defined in `Form1.Designer.cs` and controlled by event handlers in `Form1.cs`.

{
    daikinPartN_tbl.Columns.Clear();
    
    // Add Print button column
    DataGridViewButtonColumn btnColumn = new DataGridViewButtonColumn();

    btnColumn.Name = "Print";

### Grid Event Handler

**CellClick Event** (Form1.cs:169-178):

```csharp
private void daikinPartN_tbl_CellClick(object sender, DataGridViewCellEventArgs e)
{
    if (e.ColumnIndex == 0 && e.RowIndex >= 0)
    {
        string supplierPartNumber = daikinPartN_tbl.Rows[e.RowIndex]
            .Cells["SupplierPartNumber"].Value.ToString();
        string daikinPartNumber = daikinPartN_tbl.Rows[e.RowIndex]
            .Cells["DaikinPartNumber"].Value.ToString();
        
        HandlePrint(supplierPartNumber, daikinPartNumber);
    }
}
```

Detects clicks on the Print button (column index 0) and extracts row data.

## Window Dragging Implementation

Custom window movement is implemented through mouse tracking:

```csharp Form1.cs:81-93
private void moveFrm(MouseEventArgs e)
{
    if (e.Button != MouseButtons.Left)
    {
        posX = e.X;  // Record position
        posY = e.Y;
    }
    else
    {
        Left = Left + e.X - posX;  // Move form
        Top = Top + (e.Y - posY);
    }
}
```

This method is called by:
- `pnlTop.MouseMove` event
- `lblTitle.MouseMove` event
- `pcbPichonKun.MouseMove` event
