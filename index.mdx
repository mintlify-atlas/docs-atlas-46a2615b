## Overview

Daikin Etiquetas is a warehouse support application designed to streamline the process of searching and printing Daikin part number labels. Built with .NET 8.0 Windows Forms, it provides warehouse staff with a fast, reliable tool for matching supplier part numbers to Daikin part numbers.

## Key Features

<CardGroup cols={2}>
  <Card title="Part Number Search" icon="magnifying-glass">
    Search Daikin part numbers by entering supplier part numbers with real-time API integration
  </Card>
  <Card title="Label Printing" icon="print">
    Print labels for matched parts directly from the application interface
  </Card>
  <Card title="Data Grid Display" icon="table">
    View search results in an intuitive, responsive data grid with multiple matched parts
  </Card>
  <Card title="Custom Interface" icon="palette">
    Dark-themed, modern UI designed for warehouse environments
  </Card>
</CardGroup>

## Why Daikin Etiquetas?

<Note>
Built specifically for Daikin warehouse operations, this application integrates directly with the warehouse support dashboard API to provide accurate, up-to-date part number matching.
</Note>

- **Fast Lookups**: Query the warehouse support dashboard API instantly
- **User-Friendly**: Simple, clean interface requiring minimal training
- **Reliable**: Built on .NET 8.0 with robust error handling
- **Efficient**: Reduces time spent on manual part number lookups
